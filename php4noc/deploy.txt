PHP - A Simple HTML Form  The example below displays a simple HTML form with two input fields and a submit button: Example <html> <body>  <form action="welcome.php" method="post"> Name: <input type="text" name="name"><br> E-mail: <input type="text" name="email"><br> <input type="submit"> </form>  </body> </html>  Run example »  When the user fills out the form above and clicks the submit button, the form data is sent for processing to a PHP file named "welcome.php". The form data is sent with the HTTP POST method.  To display the submitted data you could simply echo all the variables. The "welcome.php" looks like this: <html> <body>  Welcome <?php echo $_POST["name"]; ?><br> Your email address is: <?php echo $_POST["email"]; ?>  </body> </html>  The output could be something like this: Welcome John Your email address is john.doe@example.com  The same result could also be achieved using the HTTP GET method:
